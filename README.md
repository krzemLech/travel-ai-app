# Travel AI App

Course project for 10xDevs

## Description

The aplication allows to create personalized trip plans for registered users

## Flow

1. User registers

- personal data
- general travel preferences

2. User has acces to his traavel plans
3. User can add a new plan by filling a form in one of two options:

- specific fields with required information set
- one big text-area of text and a model checks and returns the content in form of a json, in line with the specific form

4. After the data is submitted, the plan is generated
5. User is given a plan. Now he can:

- edit it
- print it

6. User also has access to a plan list
